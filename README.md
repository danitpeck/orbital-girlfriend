# Orbital Girlfriend 💫

a whimsical little 2D browser game where your orbiting orb girlfriend protects you by bonking enemies. it's cute. it's dumb. we love it.

## how to play

- **WASD** to move your little circle around
- your girlfriend (the yellow orb) orbits you automatically
- she bonks enemies (red circles) on contact
- don't get swarmed!!

## running locally

```bash
npm install
npm run dev
```

then open http://localhost:5173

## building

```bash
npm run build
```

static files land in `dist/` — deploy anywhere.

## tech

vanilla JS + HTML5 canvas, bundled with Vite. no frameworks, no libraries, just vibes and math.
